------------------- 1. Installation / désinstallation / mise à jour

	Pour installer ce programme, exécutez simplement le fichier "Setup_TDCSKI_vx.x.x.x" qui se trouve dans le
fichier zip que vous venez de de télécharger. 
	
	La désinstallation s'effectue via la menu démarrer ou le panneau de configuration de Windows.

	Les mises à jour sont entièrement automatiques, et vous seront proposées lorsque le programme détecte
une nouvelle version en ligne.


------------------- 2. Utilisation

	Pour utiliser le TDCSKI, il suffit de le lancer. Aucune action particulière n'est requise de votre part, 
tout le processus est automatisé, et, en cas de problème, de doute, ou de danger pour vos fichiers, le 
TDCSKI interrompt toute activité et vous propose d'envoyer un rapport de bug directement via son interface.
C'est comme MMA, zéro souci, zéro tracas ;þ

	Les missions des TDC seront accessibles dans le répertoire de missions par défaut de A10, dans un sous-
dossier TDC, lui même partagé entre les missions "SOLO" et "MULTI".

	IMPORTANT: des problèmes peuvent survenir avec votre firewall/antivirus. Un cas a déjà été vérifié avec
l'antivirus "Avira". En cas de souci, essayez de lancer le TDCSKI après avoir désactivé votre antivirus/firewall
